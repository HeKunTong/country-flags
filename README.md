## 依赖安装
```
yarn install
```

### 本地开发启动
```
yarn dev

```

### 编译打包
```
yarn build
```

### 目录

## 目录规范

```
project
|
└──src  核心目录
    | 
    ├── api    api接口
    | 
    ├── assets  静态资源 比如css、font、images 等资源文件
    | 
    ├── components  组件
    | 
    ├── config  配置 比如菜单按钮、表单
    | 
    ├── pages  页面
    | 
    ├── utils  工具类 比如request请求
    |   
    └── router  路由信息
```
